##  Descripción del proyecto

Este proyecto consiste en el desarrollo de una **Hoja de Vida (CV) interactiva** utilizando **HTML, CSS y JavaScript**, aplicando los principios de semántica, separación de responsabilidades y diseño responsivo.  
La página incluye una presentación personal, formación académica, proyectos, habilidades y enlaces a redes profesionales.

---

## Funcionalidades implementadas

- **HTML semántico:** uso de etiquetas `<header>`, `<main>`, `<section>`, `<footer>`.  
- **CSS responsivo:** uso de Flexbox, variables CSS y diseño centrado.  
   **JavaScript interactivo:**  
  - Botón para mostrar/ocultar la sección del perfil.  
  - Efecto visual al pasar el mouse sobre las habilidades.  
   **Foto de perfil** y estructura dentro de un contenedor principal
